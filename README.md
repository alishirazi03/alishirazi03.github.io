## 📋 About

This is the source code for my personal portfolio website. I'm a BS Biosciences graduate from COMSATS University Islamabad transitioning from wet-lab molecular biology to computational genomics and bioinformatics.

The website showcases:
- 🧬 Research projects (SARS-CoV-2 genomics, ATP8B1 genetic variants)
- 📝 Blog posts on bioinformatics tools and tutorials
- 🎓 Academic background and certifications
- 💼 Professional experience and skills
- 📊 Presentations and publications
